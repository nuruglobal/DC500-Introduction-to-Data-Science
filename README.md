# DSC 500 – Introduction to Data Science

## Overview

This repository documents my learning journey through DSC 500 – Introduction to Data Science at Bellevue University as part of the Master of Science in Data Science program.

The repository contains course exercises, research, exploratory data analysis, project milestones, visualizations, and practical applications of data science concepts, ethics, statistics, and business analytics.

## Course Topics

- Data Science Fundamentals
- Data Science Process
- Data Preparation and Data Quality
- Data Analysis and Statistics
- Data Visualization
- Big Data Concepts
- Data Mining
- Machine Learning Fundamentals
- Artificial Intelligence
- Data Ethics
- Business Applications of Data Science
